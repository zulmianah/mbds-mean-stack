# MBDS_Madagascar2021_backend
Backend NodeJS Mongoose pour le cours angular MBDS Madagascar 2021

## Etudiantes

- Anah Zulmianah

## Fonctionnalités

- Auhentification JWT
- Création de compte par l'admin
- Connection de l'utilisateur
- Gestion des roles : Admin, Prof, Eleve.
- Gestion des notes
- Création d'assignement par classe.

Pour le front :

```sh
    npm install
    npm start
```

Pour le back :

```sh
    npm install
    npm start
```
Le back démarre sur le port . Vérifiez qu'il soit libre.
